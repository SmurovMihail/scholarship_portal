# scholarship_portal
Обязательное программное обеспечение:
Python версии 3.12.0
PostgreSQL версии 17

Аппаратные требования:
Оперативная память: не менее 2 ГБ
Свободное место на диске: не менее 500 МБ
Доступ в интернет

Предварительные требования:
Установка Python 3.12.0
Установка PostgreSQL 17
 
Установка и запуск:
Откройте командную строку (CMD)
Перейдите в каталог backend проекта: cd [путь_к_проекту]/backend
Выполните последовательно следующие три команды:
•                 pip install -r requirements.txt
•                 python create_database.py
•                 python app.py
 
Система готова к использованию! Для начала работы перейдите в браузере по адресу: http://127.0.0.1:5000
